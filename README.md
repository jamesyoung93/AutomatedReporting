# AutomatedReporting
This app takes a dataset of predetermined structure and automatically creates a downloadable "dashboard" style report. The app also allows further interactive investigation of data in addition to the automated report. View the video demonstration at https://foretodata.com/blog-2/. To see the app go here https://jamesyoung.shinyapps.io/AutomatedReporting/ (automated report wont work for dataset with improper headings but graphing still should on web page).
